# Sana's Bookstore Website

This is a simple bookstore website built with HTML, CSS, and JavaScript.

## Table of Contents

## Features

- Browse curated book collection
- Live search functionality for books
- Footer with contact info and relevant links
- Social media integration
- Responsive design
